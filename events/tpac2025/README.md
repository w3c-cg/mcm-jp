## 概要

- 放送、出版、イベント情報など業界横断でのメディアコンテンツのメタデータの活用に関する実践的な取り組みを共有するセッション。
- TPAC 2023におけるBreakout (https://www.w3.org/2023/09/13-cross-industry-media-content-minutes.html)を受けて設立された、MCM-JP CGの取り組み成果を紹介。
- 実際に各業界で運用されているメタデータやノウハウを組み合わせることで実現可能な業界連携サービスの実装事例のデモンストレーションを実施した。
- 実践的な協調領域の明確化、検討促進・仲間づくりに向けたアイディア、W3C各グループとの役割分担などについて議論した。

## 議論における主なコメント

- 書籍のタイトルなど出版社から書店までが利用する流通用メタデータが他業界で有用であることが確認できた。一方で、キャラクター名やストーリーなどの詳細な情報を利用していくためには、構造化や権利の明確化などが必要であることが分かった。
- 出版メタデータはもともと、他業界との連携を想定して設計されたものではないが、CGの取り組みには出版業界のアプローチを変革できる可能性がある。
- メタデータの意味論と記述の不整合という従来課題が、AI活用により改善される可能性の指摘。
- CGで検討される仕組みの一般化が、国際産業にも寄与する期待。
- 欧州放送規格（DVB）における、データの構造化などとの連携の可能性がある。

## まとめと今後の方針
MCM-JP CGでは、日本国内の具体的な産業事例を収集し、想定サービス事例の実現性について検証を進める。
Media & Entertainment IG や Publishing BGなどの関連グループと連携を継続する。
